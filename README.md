# Saber Email Validator

Validador e sanitizador de emails para Windows.

## Downloads

| Arquivo | Descricao |
|---------|-----------|
| [SaberEmailValidator_Setup_v1.0.0.exe](https://github.com/saberappsrepo/SaberEmailValidator/releases/download/v1.0.0/SaberEmailValidator_Setup_v1.0.0.exe) | Instalador (Inno Setup) |
| [SaberEmailValidator_Portavel_v1.0.0.zip](https://github.com/saberappsrepo/SaberEmailValidator/releases/download/v1.0.0/SaberEmailValidator_Portavel_v1.0.0.zip) | Versao portatil (extrair e executar) |

## Requisitos

- Windows 10 ou superior (64-bit)
- Nenhuma dependencia externa necessaria

## Instalacao

### Via instalador
Execute SaberEmailValidator_Setup_v1.0.0.exe e siga as instrucoes.

### Via portatil (recomendado se o antivirus bloquear)
1. Baixe SaberEmailValidator_Portavel_v1.0.0.zip
2. Extraia para uma pasta de sua escolha
3. Execute SaberEmailValidator.exe

## Sobre

Saber Email Validator e uma ferramenta desktop para validacao e sanitizacao de listas de emails.

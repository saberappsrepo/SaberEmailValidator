# Saber Email Validator

Validador e sanitizador de e-mails para Windows.

## Download

Baixe o instalador na pagina de [Releases](https://github.com/saberappsrepo/SaberEmailValidator/releases).

## Instalação

### Método 1: Instalador (recomendado)

1. Baixe `SaberEmailValidator_Setup_v1.0.0.exe`
2. **Clique com o botão direito** no arquivo → **Executar como administrador**
3. Se o antivírus bloquear a instalação:
   - **Windows Defender:** Adicione a pasta onde o instalador está nas exclusões
     (Windows Security → Virus & threat protection → Manage settings → Exclusions → Add folder)
   - **Outros antivírus:** Adicione o instalador ou a pasta nas exceções
4. Após adicionar a exclusão, execute o instalador novamente como administrador

### Método 2: Versão portátil (sem instalação)

1. Baixe `SaberEmailValidator_Portavel_v1.0.0.zip`
2. Extraia para qualquer pasta
3. Execute `SaberEmailValidator.exe`
4. Nenhuma instalação necessária — funciona direto

### Método 3: Desativar antivírus temporariamente

Se preferir, desative o antivírus durante a instalação e reative após concluir.

## Requisitos

- Windows 10 ou superior (64-bit)
- Nenhuma dependência externa necessária
